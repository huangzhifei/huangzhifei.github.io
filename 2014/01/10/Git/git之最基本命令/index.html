发表于 2014-01-10 | 分类于 git常用命令与技巧 |

会持续更新……

建议大家直接看 git 的官方翻译 https://git-scm.com/book/zh/v1

使用 git –help 会显示出所有的命令，这篇文章存在的目的是为了更好的说明其中一些常用的命令。

其实最好的方法是自己在 github 上面创建一个项目，把命令过一遍，其实在你使用 git 命令的时候，控制台都有相应的提示。
